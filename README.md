# TINFOIL

**The AI that finds the conspiracy connecting any two things. Every source is real.**

Built for [ElevenHacks 2026](https://hacks.elevenlabs.io/hackathons/0) — Track #1: Firecrawl.

> **Disclaimer:** This project is a hackathon entry built purely for entertainment and educational purposes. All "conspiracy theories" are AI-generated fiction. The cited sources are real, but the interpretations connecting them are intentionally absurd and paranoid. This is satire — not misinformation. Please don't actually believe any of it.

## What it does

Enter two completely unrelated topics. Tinfoil will:

1. **Research** — Firecrawl searches the web for real facts about both topics
2. **Generate** — OpenAI GPT-4o weaves those facts into a dramatic conspiracy theory (streaming)
3. **Broadcast** — ElevenLabs TTS reads it aloud like a late-night paranoid radio show

Every cited fact is verifiable. Only the interpretation is paranoid.

## How it works

```
User Input → Firecrawl Web Search (8 parallel queries, ~30 sources)
           → OpenAI GPT-4o Streaming (paranoid radio host persona)
           → ElevenLabs TTS (Brian voice, streams after first 3 paragraphs)
```

- **Research phase**: 8 parallel Firecrawl queries covering history, controversies, cross-references, and direct connections between the two topics
- **Generation phase**: GPT-4o streams 15-25 paragraphs with inline `[CITE: url]` markers. Sources appear on the evidence board in real-time as they're cited
- **Broadcast phase**: TTS fires after the first 3 paragraphs — audio plays while text continues streaming. A second TTS batch handles the remaining text for seamless playback

## Tech stack

- **Next.js 15** (App Router, React 19)
- **Firecrawl** — web search API for real-time research
- **OpenAI GPT-4o** — streaming text generation with SSE
- **ElevenLabs TTS** — text-to-speech with Brian voice (`eleven_turbo_v2_5`)
- **Framer Motion** — animations
- **Tailwind CSS v4** — CRT monitor aesthetic with scanlines, vignette, beveled 80s buttons

## Running locally

```bash
git clone https://github.com/debgotwired/tinfoil.git
cd tinfoil
npm install
```

Create `.env.local`:

```
FIRECRAWL_API_KEY=your_key
OPENAI_API_KEY=your_key
ELEVENLABS_API_KEY=your_key
```

```bash
npm run dev        # http://localhost:3002
```

## Testing

```bash
npx playwright test                    # All 16 tests (15 UI + 1 full flow)
npx playwright test --grep-invert "full flow"  # UI tests only (no API calls)
```
